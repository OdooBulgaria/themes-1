Latest and original version (Native)

All native bootswatch themes refer to the themes found at bootswatch.com. 

Example: Theme Darkly

Click on the "Bootswatch Native" to open the themes.
Click on the Darkly Theme "Apply" button.

A message "Native Bootswatch Theme Changed!" is displayed.
Click on the button "your homepage" to see the result.

"That's all".

Remark:

The Darkly Theme source used is the bootstrap.css style sheet that can be found at https://bootswatch.com/darkly/bootstrap.css
